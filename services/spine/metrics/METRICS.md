## The promotion rule

The model serves only when there are at least 10
recorded journeys AND it beats the baseline on held out data. The rule lives
in `src/eta/train.ts` and serving code reads only the verdict in
`metrics.json`; nothing else decides. The promotion path itself is proven in
`test/eta-train.test.ts` on a synthetic rush hour fixture where the model
wins and is promoted.
